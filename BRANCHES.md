# 分支说明

## 主要分支

### main
- **用途**: 生产环境分支
- **保护**: 需要通过 PR 合并
- **部署**: 自动部署到生产环境

### develop
- **用途**: 日常开发分支
- **创建时间**: 2026-01-03
- **基于**: main 分支
- **说明**: 所有日常开发工作在此分支进行，开发完成后通过 PR 合并到 main 分支

## 工作流程

1. 从 `develop` 分支创建功能分支
2. 在功能分支上进行开发
3. 开发完成后提交 PR 到 `develop` 分支
4. 测试通过后，从 `develop` 创建 PR 到 `main` 分支
5. `main` 分支自动部署到生产环境

## 分支管理

- **功能开发**: `feature/功能名称`
- **Bug 修复**: `fix/问题描述`
- **文档更新**: `docs/文档名称`
- **AI 辅助**: `copilot/任务描述`
