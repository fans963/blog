## 项目起源

作为一个对设计和用户体验有追求的开发者，我决定从头构建一个个人博客网站。这个项目不仅仅是为了记录技术笔记，更是为了实践 Material Design 3（MD3）设计理念，探索现代前端技术栈的最佳实践。

**My Blog** 由此诞生 —— 一个基于 Astro、React 和 Tailwind CSS 构建的 Material Design 3 风格博客网站。

## 技术栈选择

### 为什么选择 Astro？

在选择框架时，我对比了 Next.js、Gatsby 和 Astro。最终选择 Astro 的原因：

1. **零 JS 默认**：Astro 默认不发送 JavaScript，只在需要时 hydration，极大提升了性能
2. ** Islands 架构**：可以混合使用 React、Vue、Svelte 等框架，按需加载
3. **内容优先**：完美适配博客这类内容密集型网站
4. **TypeScript 原生支持**：开箱即用的类型安全

### React + Tailwind CSS 的组合

- **React**：用于需要交互性的组件，如 ColorPicker、ThemeToggle
- **Tailwind CSS**：快速实现 MD3 设计系统，通过 `@theme` 配置自定义属性

## Material Design 3 设计实践

### 动态颜色系统

MD3 的核心特性之一是动态颜色。网站实现了完整的动态颜色切换功能：

```typescript
import { applyThemeToDom } from '../utils/theme';

function changeTheme(color: string) {
  const theme = createThemeFromColor(color);
  const cssVariables = themeToCssVariables(theme);
  applyThemeToDom(cssVariables);
}
```

用户可以：
- 选择 8 种预设颜色
- 使用颜色选择器自定义主题色
- 切换深色/浅色模式
- 所有设置自动保存到 localStorage

### MD3 组件实现

网站实现了多个 MD3 核心组件：

- **Filled Button**：主要操作按钮
- **Tonal Button**：次要操作按钮
- **Icon Button**：图标按钮，用于导航栏
- **Chips**：分类筛选标签
- **Cards**：文章卡片，带 hover 动效

### 动效设计

遵循 MD3 的 Motion 规范：

```css
--md-motion-easing-standard: cubic-bezier(0.2, 0, 0, 1);
--md-motion-duration-short: 200ms;
--md-motion-duration-medium: 400ms;
```

网站在所有设备上都有良好表现：

- **桌面端 (≥1024px)**：完整三栏布局
- **平板端 (640px - 1024px)**：双栏布局
- **手机端 ()**：单栏布局，导航栏自适应

## 开发过程中的收获

### 1. TypeScript 类型安全

通过 Astro 的内容集合（Content Collections）功能，实现了文章数据的类型安全：

```typescript
const blog = defineCollection({
  schema: z.object({
    title: z.string(),
    description: z.string(),
    date: z.date(),
    category: z.string(),
    tags: z.array(z.string()),
    image: z.string().optional(),
    featured: z.boolean().default(false),
  }),
});
```

### 2. 组件设计模式

采用 Atomic Design 思想，将组件分为：
- **Atoms**：按钮、图标、标签
- **Molecules**：搜索框、分类卡片
- **Organisms**：导航栏、文章卡片
- **Templates**：页面布局

### 3. CSS 架构

使用 Tailwind CSS 的 `@theme` 功能，定义完整的 MD3 设计令牌：

```css
@theme {
  --color-md-primary: rgb(var(--md-sys-color-primary));
  --color-md-on-primary: rgb(var(--md-sys-color-on-primary));
  --color-md-surface: rgb(var(--md-sys-color-surface));
  /* ... 更多变量 */
}
```
