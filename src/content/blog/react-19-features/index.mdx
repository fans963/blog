React 19 引入了许多令人兴奋的新特性。

## 主要新特性

### 1. 编译器优化

React Compiler 自动优化重新渲染，减少不必要的计算。

### 2. use hook

新的 `use` 钩子允许在条件语句中使用 hooks。

### 3. Actions

简化表单处理和数据提交。

## 升级建议

建议逐步升级，先在非关键项目上测试。
