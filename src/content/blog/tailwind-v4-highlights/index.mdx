Tailwind CSS v4 是一个重大更新，带来了许多改进。

## 主要变化

### 1. 原生 CSS 配置

现在可以直接在 CSS 中配置主题：

```css
@theme {
  --color-primary: #0066cc;
  --font-sans: 'Inter', sans-serif;
}
```

### 2. 性能提升

v4 编译速度更快，生成的 CSS 更小。

### 3. 改进的暗色模式

更简单的暗色模式支持。
