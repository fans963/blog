TypeScript 最佳实践总结。

## 1. 启用严格模式

```json
{
  "compilerOptions": {
    "strict": true
  }
}
```

## 2. 使用类型推断

避免不必要的类型注解，让 TypeScript 自动推断。

## 3. 使用联合类型和交叉类型

合理使用类型组合，提高代码的灵活性。

## 4. 泛型

使用泛型编写可复用的组件和函数。
